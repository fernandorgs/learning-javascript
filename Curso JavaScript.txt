AULA 1 - Variáveis e Constantes:

CTRL+Shift+i = Modo inspeção no navegador (ou botão direito, inspecionar)
Console
Nos '...' pode alterar a posição do console
CTRL+l = limpar

; no final da linha é opcional

- var cria variável (ex: var tituloDoConteudo = "Variáveis e Constantes";)
- comando let é muito similar ao var (e não foi explicado)
- const = constante (const duraçãoMaximaDoCurso = "12 meses";)

Formato de escrever uma variável = Camel Case = começa tudo em minuscula, e depois vai colocando maiuscula na primeira letra de cada nome, não usa espaço e não começa com numeros

AULA 2 - Tipos Primitivos
String
var nomeEstudante = "Ana";
var nomeEstudante = 'Ana';

Boolean
var aprovada = true;
var aprovada = false;

Number
var idade = 12
var custo = 350.90

Comando typeof identifica o tipo de variavel ou valor
exemplo: 
typeof 14
"number"
typeof aprovada
"boolean"
